# Iterative_Reweighting_Weighted_Least_Squares_for_Rank_Aggregation

Under BTL_Conditioned_with_Deterministic_Outcomes folder
It contains the source codes and MATLAB files for the adaptively modified weighted least squares algorithm and the other base algorithms to compare performance under handle sparse, incomplete, and imbalanced graph structures under  BTL conditioned setup with limited deterministic outcomes .

Under BTL_Conditioned_with_Non_Deterministic_Outcomes folder
It contains the source codes and MATLAB  files for the adaptively modified WLS algorithm and the other base algorithms to compare performance under  BTL conditioned setup with limited deterministic outcomes where outcomes include a Bernoulli noise.

Under Additional_Convergence_With_Deterministic_Outcomes folder
It contains the source codes and MATLAB  files for the adaptively modified WLS algorithm and standard WLS for the analysis of correlation measures including Top-k, True rank of inferred top-1, Estimated rank of true top-1 and Spearman footrule distance under BTL conditioned deterministic setup. 

Consider that the main.m files include the mainfile to call the functions and run the setup.
